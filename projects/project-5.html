[Spring Boot] 헬스장 커뮤니티 팀프로젝트

헬스장을 선택할 때 회원권금액, PT금액과 같은 정보를 얻으려면
대부분 직접방문 또는 전화문의를 해야하는 불편이 따릅니다.
이 점에 착안하여, 서울 지역별 헬스장에 대한 회원들의
솔직한 후기공유를 통해 다양한 정보를 얻을 수 있는
사이트를 제작하였습니다.
프로젝트 상세설명은 깃허브 저장소(Github repository) README에 작성하였습니다.

Frontend

: HTML5, CSS3, JavaScript, JQuery, Ajax, bootstrap, thymeleaf

Backend

: Java, JSP, Spring boot, Gradle, Spring-Data-JPA, Spring-Security

DB

: Maria db

WAS

: Apache Tomcat
